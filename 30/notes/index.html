Increments and Decrements

Increments are adding somethign to the value. It can be added via let i++. Remember, the order of actions of i++:

  1. Console log will show the result of i.
  2. The increment ++ will be added and the value will be saved in the memory.

If we add the increment before the variable: let ++i, the order of actions will bee different:

  1. The increment ++ will be added and the value will be saved in the memory.
  2. Console log will show the result of i.

Decrements are deducting somethign from the value. let i--.

  • Increments (and decrements) won't work with the text in the strings, will return NaN.
  • Increments (and decrements) will work with the number in the strings.
  • If the variable is empty string, the counter would use it as 0.

Loops

There are 3 loop types in JS:

  1. for
  2. while
  3. do while

for should have 3 parts: the beginning, the condition, and the step. It can also work without the first and the last, however, we need to use it like for(; 1+1;), where ; will represent the missing parts.

The variable that is declared outside of the loop, and changes in the nested loop can be seen outside of the loops later.

while should have 1 part: the condition, and inside the loop, should have stepper. it's more common to use while with decrement.

do while will execute something at least once. Works similar to while. Used rarely.
