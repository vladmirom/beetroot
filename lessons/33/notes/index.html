Arrays

Arrays can be created with the Array constructor or as Literal array

Arrays in js are like the unassociative arrays in php. Associative arrays are Objects. Arrays have index and value. Functions are not usually stored in arrays.

We can add values to arrays this way:

  • fruit[1] = 'orange';
  • fruit[fruit.lenght - 1 ] = 'banana'; adds to the last element of the array.
  • fruit.at(-1) = 'clenentine'; adds to the last element of the array.
  • Add values to array via methods.

There are various methods in arrays. Here are some of them:

  • fruit.length() calculates the length of the array.
  • delete fruit[0] deletes the value of the element in array, leaving the index.It will be [0] => empty
  • fruit.splice( 2, 1) will cut the element/s from array and return them. We need to add a starting index and number of elements that we want to remove as parameters. It also can negative first parameter, to start counting the index from the end of the array.
  • fruit.splice( 2, 0, 'melon', 'watermelon'); will add 2 more values after the s index.
  • fruit.slice( 1, 3) Will cooy values into the new array, without affecting the first array.
  • fruit.concat(['pear', 'peach'], ['lemon', 'apple']) will cooy values into the new array, and add to the end of fruit array 2 more arrays. Does not create unique values. So the new array can have 2 'apple'.
  • fruit.find(function(item, index, array) { }) will find some value in array.
  • fruit.findIndex((item => item === 'apple')); will return the index of the finded 'apple'.
  • fruit = fruit.map((item => item + ' juice')); will iterate through the whole array and do something with each value.
  • Array.isArray(fruit) {} will check if the array is array, returns boolean.
  • fruit.indexOf()
  • fruit.lastIndexOf()
  • fruit.includes()
  • fruit.filter()
  • fruit.sort()
  • fruit.reverse()
  • fruit.split()
  • fruit.join()

We can receive the values from arrays this way: fruit[0];.

We can clone objects and merge 2 and more objects into one with either for or Object.assign()

There are 4 ways to compare objects:

  • Referential equality. We can use ===, ==, and Object.is(). Will give true only if they are of the same instance (like cloned objects), otherwise - false.
  • Manual comparison. Just comparing manually the values of the property of one and another object: object1.name === object2.name.
  • Shallow equality. It is a bit similar as a manual comparison, but a bit more complex and using Object.keys(). Checks only 1 level of object properties.
  • Deep equity. It is similar to shallow equity, but goes deeper and compares the deeper levels of object properties.
There is also a trick to compare 2 objects using JSON.stringify(). The idea is to convert objects into strings and compare those.
