Arrays

Arrays can be created with the Array constructor or as Literal array

Arrays in js are like the unassociative arrays in php. Associative arrays are Objects. Arrays have index and value. Functions are not usually stored in arrays.

Adding values to arrays:
  1. fruit[1] = 'orange';
  2. fruit[fruit.lenght - 1 ] = 'banana'; adds to the last element of the array.
  3. fruit.at(-1) = 'clenentine'; adds to the last element of the array.
  4. Add values to array via methods.
Array methods (some of them):
  • fruit.length() calculates the length of the array.
  • delete fruit[0] deletes the value of the element in array, leaving the index.It will be [0] => empty
  • fruit.splice( 2, 1) will cut the element/s from array and return them. We need to add a starting index and number of elements that we want to remove as parameters. It also can negative first parameter, to start counting the index from the end of the array.
  • fruit.splice( 2, 0, 'melon', 'watermelon'); will add 2 more values after the s index.
  • fruit.slice( 1, 3) Will cooy values into the new array, without affecting the first array.
  • fruit.concat(['pear', 'peach'], ['lemon', 'apple']) will cooy values into the new array, and add to the end of fruit array 2 more arrays. Does not create unique values. So the new array can have 2 'apple'.
  • fruit.find(function(item, index, array) { }) will find some value in array.
  • fruit.findIndex((item => item === 'apple')); will return the index of the finded 'apple'.
  • fruit = fruit.map((item => item + ' juice')); will iterate through the whole array and do something with each value.
  • Array.isArray(fruit) {} will check if the array is array, returns boolean.
  • fruit.indexOf() parces the array, searches for the value mentioned in parameter, and returns the index of the FIRST found value, or -1 If the value is not found. Can parse a string.
  • fruit.lastIndexOf() parces the array, searches for the value mentioned in parameter, and returns the index of the LAST found value, or -1 If the value is not found. Can parse a string.
  • fruit.includes() defines if the value exists in array. Returns boolean.
  • fruit.filter() filter all elements with the function f. It it returns true, the element will be added to the returned array. These filters will be handy: inBetween(a, b) - between a and b, including them, and inArray([...]) - the values that are in this array.
  • fruit.sort(fn) sorts elements in array in ascending order. Method converts values into the strings and then compares them. It is possible to create own sorting algorithms.
  • fruit.reverse() reverses the order of values in array and returns it.
  • fruit.split(delim) divides an array by the delimisher from the parameter.
  • fruit.join(glue) converts array into one string by adding all the values using a joint from the parameter.

Getting values from the array.

We can receive the values from arrays this way: fruit[0];.

Looping through array:
  1. With the help of for loop. The classic example: for (let i = 0; i < fruit.length; i++) { do something with every element in array; }.
  2. With the help of for of loop. Works well with arrays or strings. The example: for (let char of 'test') { do something with char variable; } and it will work 4 times: 't', then 'e', then 's', and the last one 't'.
  3. With the help of forEach(). We can get single item, index and the general array in it, which is quite handy.
  4. The object iterator for in will work also, since array can be considered as object. However, this method works more slow than for of, and not commonly used.
  5. With the help of methods: map(), filter(), find()
