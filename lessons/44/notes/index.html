Objects

Objects can be created with the Object constructor or as Literal object

Objects have pairs (properties): key and value. Functions in users are called methods.

We can check if there is key in object using the directive in.

It is important to access object property by key in for. for (key in user) { (user[key]); }.

We can clone objects and merge 2 and more objects into one with either for or Object.assign()

There are 4 ways to compare objects:

  • Referential equality. We can use ===, ==, and Object.is(). Will give true only if they are of the same instance (like cloned objects), otherwise - false.
  • Manual comparison. Just comparing manually the values of the property of one and another object: object1.name === object2.name.
  • Shallow equality. It is a bit similar as a manual comparison, but a bit more complex and using Object.keys(). Checks only 1 level of object properties.
  • Deep equity. It is similar to shallow equity, but goes deeper and compares the deeper levels of object properties.
There is also a trick to compare 2 objects using JSON.stringify(). The idea is to convert objects into strings and compare those.

const can be changed with objects.
