Cookies

We can add remove and read cookies. Secured cookies can not be reemoved.

Can be found in Devtools (inspect) > Application > Cookies.

Path is localizing cookies. Path: / is for all the weebsite. Path: /about is for about page only.

Cookies are only available on 1 website. It depends on the domain name. If we haave iframe with another website cookies won't be accessible.

Cookies live within the session. They also can be set to expire.

Size of the cookie: it depends on value.

Check cookies: document.cookie. We can run it right in console

Sessions

Sessions are visible only in one tab. If you switch tabs session won't work.

Can be found in Devtools (inspect) > Application > Local Storraage.
