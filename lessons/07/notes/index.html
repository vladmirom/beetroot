Notes for the lesson 7. CSS selectors.

General selectors

  1. * star selector selects all elements on the webiste. Useful to clear and prepare your css for the further work. It also can be used in nesting (for instance, we don't have access to DOM structure).
  2. Dont use anything inside the * except margin, padding. Font size, font weight, lineheight should not be set here. Otherwise, the correct nesting will break here.
  3. We can work directly with tags. Not the best practice.
  4. We can use classes and ids in css. Id has bigger specificity, so id will overwrite class.
  5. We don't style id because it is unique for the page, and classes can be reused on the page.
  6. We don't write px to 0.
  7. The level of nesting is defined by space between elements. This code div h3 means to select every h3 element that is located inside the div.
  8. If two classes or elements are added without the space css will look for them as sibings. EG: div .class-red.class-blue means to select the element inside the div that has both class-red class-blue classes.
  9. The code writen in the bottom will overwrite the code in the top and will be used.

General styling

  1. It's better to add font size, weight and line-space in body
  2. Font is set or duplicated in html tag to set rems. The defaut font size is 16px.
  3. line-height is better set in % if font-size is in rem or em.

CSS operators

  1. Nice reference is here: Operators
  2. > tells to find child element. EG: .some-class > p will select every p element inside the some-class. In this case <div class="some-class"><div><p>some text here</p></div></div> it will not be selected. In this case <div class="some-class"><p><p>some text here</p></p></div> it will be selected, because it inherits father's properties.
  3. + selects all the siblings of the element, except this element.
  4. line-height is better set in % if font-size is in rem or em.
