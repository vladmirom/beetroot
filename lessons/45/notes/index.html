Functional and Class components.

Functional components are the ones that are created with the help of functions. function App () { return('something') } or const App = () => { return('something') }

Class components are the ones that are created with the help of classes. Assigning it is a bit different than functional.

  • First, we need to import React: import React from 'react';.
  • Then, we form class class NewsItemClass extends React.Component { render() { return('something') } }
  • Also add export default NewsItemClass;

React is abandoning Class components and moving towards Functional components, so do we.

Props.

Props are variables that are used to wpass some information (like content) from parent to child. It help us to reuse the component many times.

There are several thins to do to use props in a functional component.

  • First, we add props in the parent file where we call our component <NewsItemFunc title="some title" text="some content" />.
  • Then, add props as a parameter to the function: function NewsItemFunc(props) { return('something') }.
  • The last thing is to output the props conntent using the {}: <h3>{props.title}</h3>
  • It's better to add the whole object, as one props. It will reduce the amount of code written.

There are several thins to do to use props in a class component.

  • First, we add props in the parent file where we call our component <NewsItemFunc title="some title" text="some content" />.
  • Before render method, we need to add constructor constructor(props) { super(props) }. Props will work even without it, but this is the right way to do it and some logic might be lost without it.
  • The last thing is to output the props conntent using the {}: <h3>{this.props.title}</h3>. Note, since this is class, we need to use this.

Important! We can not re-declare props inside the component where it is used. Props are created only for reading data. However, we can create a separate variable and declare it there.

If we want to change the value of props, we can use hooks and send the request form the child to father to change the value and re-render it.
