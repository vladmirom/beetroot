Slick Slider

Slick slider is working based on jQuery. So it should be installed before the plugin.

Useful description of the plugin is here: https://kenwheeler.github.io/slick/.

More useful info is here: https://github.com/kenwheeler/slick

Swiper Slider

Swiper seems to be better than slick. First of all it is not using jQuery. Secondly, it has more options.

We can take it from here: https://swiperjs.com/get-started.
