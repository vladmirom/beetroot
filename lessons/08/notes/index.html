Notes for the lesson 8. Box model.

Box model

  1. Box model is nicely described here.
  2. padding is internal spacing of the element or spacing before the border. It's good to use with the background, or background-color.
  3. margin is external spacing of the element. It's quite good to use between elements.
  4. Margin has an interesting property: margins overlap using the principle of bigger magrin. Let's say we have 2 elements, and first has margin-bottom: 100px; and second has margin-top: 100px;, the space between 2 elements will be 100px, not 200px. is external spacing of the element.
  5. Paddings don't have the same property as vertical margins, they are added one to another.
