Advanced SCSS p2. and SEO

SEO

  • <title> is important.
  • <meta name="description" content="Title"> Is the next after the title. Shows the description of the website in search results.
  • External links. Important to add tardet="_blank". We also can add rel attribute.
  • rel="noindex" - it's better to use it, but if the website is like Wikipedia and your content is relevant, then it's beneficial.
  • rel="nofollow" - prohibits to bot to go through the link.
  • rel="noreferrer" - doees not give the information about your website in the analytics of the other website.
  • rel="noopener" - blocks the external resource to open your page.
  • <h1> only once on the page.
  • Remember about semantics. It's important.
  • alt attribute always in <image>
  • <meta name="robots" content="index, follow"> - index or no index of the page.
  • robots.txt - should be in the root folder of the website. See robots.txt for this lesson.
  • Open Graph - enriching content mostly for Social Media.
  • Optimizing images. Need to add width and height to image, not for styling, but for rendering.
  • It's important to use webp.
  • It's better to use images up to 100kb.
  • Animation is bad for crawlers. Can add some delay to load all animation on the website.
