Units

  • px - Basic calculation.
  • rem - rem stands for "root em". Takes the html font size as a basis for calculation.
  • em - em stands for "em font size". Takes the parent's font size as a basis for calculation.
  • % - We can calculate on the basis of percentage.
  • vw and vh - view width and view height. The basis is the size of the screen.

Including fonts

We can add Google fonts, they aree free.

Paid fonts is a bit different story.

  1. Create folder "fonts" in assets.
  2. Upload all the font files there.
  3. Add them as a font face. Probably in a separate css file (see css file).
  4. Remember to add font-family to each font file.
