Js variables

there are 3 ways to declare the variable

  • var - can be visible everywhere in the doc.
  • let - is a variable that can be re-written. It is accessible only in the place where we have {}.
  • const - can not be redeclared.

The name can include:

  • It can have letters, digits, dollar sign, and underscore.
  • It can not have the digit first.
  • It is recommended to use latin letters, not cyrilic or hieroglyphs.
  • It is not recommended to use hyphen and numbers.
  • Register counts here: user and User are 2 different variables.
  • There are reserveed words that is not recommended to use: https://www.w3schools.com/js/js_reserved.asp

Styles of variable cases

  • myUserInfo - camel case.
  • my_user_info - snake case.
  • MyUserInfo - pascal case.
  • my-user-info - kebab case.

Basic functions:

  • Alert - just creates a message.
  • Prompt - we can get a value from here.
  • Confirm - we can get the boolean.

Types of data

There are 8 types of data. However, many sources give different number of data types

  1. Number - represents integers and floats.
    • Integers
    • Floats
    • Infinity - can be infinity or -infinity
    • NaN - "Not a number".
  2. BigInt - 2 у 53 ступені - 1. Can be with + and with -.
  3. String - has 3 ways to declare. When using sqewed parenthesis, we can add variables inside.
  4. Boolean - trye/false.
  5. null - значення пустоти. Значення присвоєне і є пустотою.
  6. underfined - значення не присвоєне.
  7. Arrays.
  8. Objects.

All types that have 1 value are named primitive, the others (object and array) are called constructives.

Function typeof helps to define the type of data. Can use with () and without.
