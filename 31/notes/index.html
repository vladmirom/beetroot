JavaScript Functions

Functions is used to create repeated code. These are some notes:

  1. function name() {} - is the way to declare function.
  2. let variables can be visible only inside the function.
  3. We can call a function in any place in the document. Either before, or after the function declaration.
  4. We can declare variable in the function and it will be used after we called the function.
  5. Functions don't support skipping the parameters. But we can add underfined variable as a first parameter.
  6. let variable = function () {} - is the way to create functional expression.
  7. Function declaration and function expression are differently used by js core. Functional expression can only be called after this expression.
  8. Anonymous functions that don't have name. They can be used in functional expression.
  9. IIFE - Immediately invoked function expression. Executes immediately anonymous expression.
  10. If we create the variable in IIFE with var someVariable = , it will be local, but it will be global if wee declare as someVariable = .
  11. We can user arrow functions, they are very handy. let caclulate = (a, b) => a + b; and we will call it as let caclulate(1, 2);.
  12. Function caan also declarared with new Function() let sayHowdy = new Function('a', 'b', 'return a + b');
