Advanced SCSS

  • It's better not to use the calc() css function.
  • In native calculation (+/-) in scss we can calculate only the same data types. (px - px, % - %, rem - rem)
  • We can multiply any data types except pixels. In this case we can use just number: 10px * 10.
  • Division: just / sign is deprecated. The division should be used in (). (100px / 2);. Also, if we divide the same data types, they will give the number without the datatype. In this case we just add the datatype additionally: (100px / 2px) + px;. Different data types don't work.
