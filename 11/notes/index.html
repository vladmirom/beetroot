BEM notes

BEM - Block Element Modifier.

Block is an independent element that can be used anywhere. We can place it anywhere on the website. Block can have othe blocks inside.

Element is not an independent element, it can not exist without the block.

Modifier is a state of the element or block.

We don't style tags in BEM. We style only classes.

We can not add modificator only: 

<div class="img-cart architectural-photography__grid--col-2-3">
it should always have the block and element class first: 
<div class="img-cart architectural-photography__grid architectural-photography__grid--col-2-3">
