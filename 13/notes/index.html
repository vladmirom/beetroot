Different types of positioning in css

  • inline - obviously, sets elements in one line one by one.
  • relative - similar to inline, but can move it with the help of top, right, bottom, left.
  • absolute - defines element relative to the body tag, or parent width and height. If top, right, bottom, left are not set, it is going to top left corner. Absolute is negating the block element principles. It has properties of inline-block.
  • fixed - getting sticked in relation to our `<html>`.
  • sticky - the element is following screen while scrolling. Parent element should have height. And the element should also havee the `top: something here;`
