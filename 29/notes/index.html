JavaScript Comparison Operators

We can use different operators to compare values:

  1. == - compares the value of variable.
  2. === - compares the value and type of variable.
  3. != - compares the value of variable. Does not equal.
  4. !== - compares the value and type of variable. Does not equal.
  5. > - greater than.
  6. < - less than.
  7. >= - greater than or equal to.
  8. <= - cless than or equal to.
  9. ? - ternary operator.
